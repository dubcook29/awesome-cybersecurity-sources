# Awesome Cybersecurity Sources [![Awesome](https://awesome.re/badge.svg)](https://awesome.re)

This is a fun and awesome project. Its purpose is to collect information sources related to **cybersecurity** from around the world, 
with the help of the GitHub community. These sources can be "company websites, product websites, forums, tech websites, blogs, podcasts, Twitter, Weibo,  etc."
—**any information source related to cybersecurity**.

The initial plan is to compile 100 interesting information sources and use these 100 sources as a sample for "archiving" (still in the process of exploration). 
Once this "archiving" is complete, I believe more people will participate.

If you have reliable sources of information, you can submit a [**ADD NEW SOURCE**](https://github.com/dubcook29/awesome-cybersecurity-sources/issues/new?template=add-new-sources.md).
If accepted, you will see it in the [**sources.csv**](#) file.
You can also submit any suggestions via [**ISSUES**](https://github.com/dubcook29/awesome-cybersecurity-sources/issues/new?template=BLANK_ISSUE), which will receive a response and be carefully considered for adoption.
